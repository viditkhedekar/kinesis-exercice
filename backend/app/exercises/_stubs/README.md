# Exercise config stubs

These are starting-point configs for the remaining seven exercises. They follow
the exact same schema as the three implemented exercises (`squat`, `bicep_curl`,
`pushup`). To activate one, tune its thresholds against real footage and move the
file up one directory into `app/exercises/`. **No engine code changes are
required** — the analysis pipeline is fully config-driven.

Thresholds below are reasonable first guesses, not validated values.
